Implement `actioneer audit --fix`.

Behavior:
- Applies all available safe fixes without prompting and without requiring `--yes`.
- Fixes mutable refs by pinning them to full SHA when target data is available.
- Fixes SHA/comment mismatches when the correct SHA is known.
- Short SHA refs are only fixable when they uniquely match a known target SHA.
- Unfixable findings remain and cause audit to fail.
- Supports `--dry-run` and respects `--mode json`.
- Patches files safely.

Output in JSON should include a `fixes` array.
`ok` is true only when no findings remain after fixes.
Keep tests green.
